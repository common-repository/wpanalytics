Integrate google analytics in wordpress

== Description ==

Integrate the google analytics code into the word press, so the changes will live through update of wordpress and also provide an setting page to easy setup and update the analytics code. 

== Installation ==

1. Upload `wpanalytic` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Paste the google analytics code in the Settings->WpAnalytics
